Installing

Toolbox

If you're downloading Donation Board 3.0 from the creator store, follow these instructions:


1

Download the model from the creator store.

2

Open Toolbox in Roblox Studio by going to Home → Toolbox

3

Find and insert Donation Board 3.0 and into workspace.




Configuring

Game Settings

For Donation Board 3.0 to work properly, we need to enable API Services:


1

Open Game Settings in Roblox Studio by going to File → Game Settings

2

Within this page, direct to the Security section.

3

Enable API Services and press Save



Developer Products

To create developer products, you'll just need to enable them in the developer products page:


1

Play Test in Roblox Studio by pressing the Play Arrow

2

Press the big Donate Button at the bottom of the donation board.

3

Press on the Developer Products button and enable/disable the products you want displayed.



Board Settings

We offer you full customizability of the board, which you can only change in studio by pressing the ⚙ icon.

1

Play Test in Roblox Studio by pressing the Play Arrow

2

Press the ⚙ icon at the bottom of the donation board to view settings.
